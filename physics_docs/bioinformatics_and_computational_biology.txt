1. WHAT IS BIOINFORMATICS?
---------------------------
Bioinformatics applies computational and statistical methods to biological data —
especially DNA, RNA, and protein sequences. Enables analysis of massive biological
datasets generated by modern high-throughput technologies.

Key Biological Databases:
  GenBank (NCBI)      : Nucleotide sequence database; >300 billion bases.
  UniProt/Swiss-Prot  : Protein sequences and functional annotation.
  PDB (Protein Data Bank): 3D structures of proteins, nucleic acids.
  Ensembl             : Annotated eukaryotic genomes.
  KEGG                : Metabolic pathways and functional annotations.
  OMIM                : Genetic disorders and gene-disease associations.
  GEO (Gene Expression Omnibus): Gene expression datasets (microarray, RNA-Seq).
  dbSNP               : Single nucleotide polymorphisms.
  ClinVar             : Variants and clinical significance.

Sequence Formats:
  FASTA: Simple text; header line (>) + sequence. Universal.
  FASTQ: Like FASTA + quality scores per base (Phred scores). Raw NGS reads.
  SAM/BAM: Aligned sequence reads (Sequence Alignment/Map; Binary form).
  VCF: Variant Call Format — genetic variants.
  GFF/GTF: Gene annotation coordinates.
  PDB format: 3D coordinates of atoms in biological structures.

1.1 SEQUENCE ALIGNMENT
-----------------------
Pairwise Alignment:
  Global (Needleman-Wunsch): Aligns entire length of both sequences. Best for
  similar-length homologous sequences.
  Local (Smith-Waterman): Finds best local alignment. Best for conserved domains
  within divergent sequences.
  Gap penalties: Penalise insertions/deletions. Scoring matrix (BLOSUM, PAM) weights
  amino acid substitutions based on evolutionary data.

BLAST (Basic Local Alignment Search Tool):
  Heuristic algorithm — fast local alignment; not guaranteed optimal.
  Query sequence → searched against database → returns homologous sequences.
  E-value: Expected number of alignments with that score by chance.
    E-value < 1e-5 = significant; < 1e-50 = very strong homology.
  Types: BLASTn (nucleotide-nucleotide), BLASTp (protein-protein),
  BLASTx (translated nucleotide → protein), tBLASTn, tBLASTx.

Multiple Sequence Alignment (MSA):
  Align 3+ sequences simultaneously. Identifies conserved residues.
  Tools: ClustalW, MUSCLE, MAFFT, T-Coffee.
  Output: Used for phylogenetic tree construction; identifying conserved functional sites.

2. NGS DATA ANALYSIS PIPELINES
--------------------------------
General NGS Pipeline:
  Raw reads (FASTQ) → Quality control (FastQC, Trimmomatic) → Alignment/Mapping
  (BWA, Bowtie2, STAR) → BAM/SAM processing (SAMtools, Picard) → Downstream analysis.

DNA-Seq (Whole Genome/Exome):
  Alignment: BWA-MEM → sorted/indexed BAM.
  Mark duplicates: Picard MarkDuplicates.
  Variant calling: GATK HaplotypeCaller → VCF → filtering (VQSR) → annotation
  (Ensembl VEP, ANNOVAR).
  Structural variants: LUMPY, Manta, DELLY.
  Copy number variants (CNV): CNVnator, GATK CNV.

RNA-Seq (Transcriptomics):
  Alignment: STAR or HISAT2 → aligned to genome.
  Quantification: featureCounts, HTSeq, or pseudo-alignment (Salmon, Kallisto — fast).
  Differential expression: DESeq2 or edgeR (negative binomial model).
    DESeq2: Size factor normalisation; shrinkage estimators for fold change.
    Output: Log2 fold change + adjusted p-value (BH correction) for each gene.
  Gene set enrichment analysis (GSEA): Are gene sets (pathways) enriched in DE genes?
    Tools: clusterProfiler, GSEA, Enrichr.

ChIP-Seq:
  Chromatin immunoprecipitation + sequencing → maps protein-DNA binding.
  After alignment: Peak calling (MACS2) → narrow peaks (TFs) or broad peaks (histones).
  Quality metrics: FRiP score (fraction reads in peaks; >0.3 recommended).
  Motif analysis: MEME, Homer → identify binding motifs in peaks.
  Differential binding: DiffBind package.

3. GENOME ASSEMBLY & ANNOTATION
---------------------------------
Genome Assembly:
  Overlap-Layout-Consensus (OLC): Short reads; Sanger era.
  de Bruijn graph: Short NGS reads; break into k-mers → assemble graph.
    Tools: SPAdes (bacteria), Velvet, SOAPdenovo.
  Long-read assembly: PacBio/Nanopore → much longer reads → simpler assembly.
    Tools: Canu, Flye, Hifiasm. Near-telomere-to-telomere assembly now possible.
  Assembly metrics: N50 (contig length at which 50% of assembly is in contigs of this
  length or longer); BUSCO (genome completeness using conserved genes).
  T2T (Telomere-to-Telomere) Consortium 2022: First truly complete human genome;
  added ~200 Mb previously missing (centromeres, rDNA arrays, segmental duplications).

Genome Annotation:
  Ab initio: Identify genes by statistical patterns (codon usage, splice sites) — Augustus, GeneMark.
  Homology-based: Map known proteins/ESTs to genome — BLAST, Exonerate.
  RNA-Seq evidence: Transcriptome data guides splice site discovery — MAKER, BRAKER2.
  Final annotation: NCBI RefSeq, Ensembl, Wormbase, FlyBase.
  ~20,000 protein-coding genes in human genome; also: lncRNAs, snoRNAs, miRNAs, pseudogenes.

4. PROTEOMICS & STRUCTURAL BIOINFORMATICS
-------------------------------------------
Mass Spectrometry-Based Proteomics:
  Bottom-up proteomics: Proteins → trypsin digest → peptides → MS/MS.
  Data-dependent acquisition (DDA) vs Data-independent acquisition (DIA).
  Database search: SEQUEST, Mascot, MaxQuant → match spectra to peptide sequences.
  Label-free quantification (LFQ) or iTRAQ/TMT labelling for relative quantification.
  SILAC: Metabolic labelling with heavy amino acids → exact quantification.
  Interactomics: AP-MS (affinity purification) → pull-down protein complexes.
  BioID/TurboID: Proximity labelling with biotin ligase → maps protein neighbourhood.

Protein Structure Prediction:
  Homology modelling: Build structure based on known structure of homologous protein.
    Tools: SWISS-MODEL, Modeller. Works when >30% sequence identity.
  Ab initio: Predict from sequence alone — Rosetta (energy minimisation).
  AlphaFold2 (DeepMind, 2021): Deep learning; near-experimental accuracy.
    Uses multiple sequence alignment (MSA) + attention-based neural network (Evoformer).
    Protein Structure Module: Places backbone + side chains iteratively.
    Trained on ~170,000 PDB structures + evolutionary data.
  ESMFold (Meta AI, 2022): Language model approach; fast single-sequence prediction.

5. MACHINE LEARNING IN BIOLOGY
--------------------------------
Supervised Learning Applications:
  Variant pathogenicity prediction: CADD, SIFT, PolyPhen-2 → predict effect of amino
  acid changes. DeepVariant (Google): CNN for variant calling from NGS.
  Splice site prediction: SpliceAI (Illumina): Deep learning predicts splicing alterations
  from sequence → identifies cryptic splice variants causing genetic disease.
  Protein function prediction: DeepGOPlus; protein-protein interaction prediction.
  Drug-target interaction: GraphDNN, DeepDTA → predict binding affinity.

Unsupervised Learning:
  scRNA-Seq clustering: PCA dimensionality reduction → UMAP/tSNE visualisation →
  k-means or Leiden/Louvain community detection → cell type identification.
  Seurat, Scanpy: Standard scRNA-Seq analysis packages. CellChat: Intercellular communication.
  Trajectory analysis (pseudotime): Monocle3, diffusion pseudotime → reconstruct
  developmental trajectories from single-cell data.

Deep Learning in Genomics:
  Enformer (DeepMind, 2021): Predicts gene expression from DNA sequence alone;
  models enhancer-promoter interactions up to 100kb away. Transformer architecture.
  DNABERT: BERT-based model pretrained on DNA sequences → fine-tuned for multiple tasks.
  Protein language models (ESM-2, ProtTrans): Trained on 250M protein sequences →
  embeddings capture evolutionary information; outperform traditional methods.

6. SYSTEMS BIOLOGY
-------------------
Systems biology models biological processes as networks of interacting components.

Gene Regulatory Networks (GRNs):
  Nodes: Genes/TFs. Edges: Regulatory relationships. Inferred from scRNA-Seq data.
  Tools: SCENIC (regulon analysis); CellOracle (TF → target gene perturbation simulation).
  Network motifs: Feedback loops (bistability), feedforward loops (filtering), cascades.

Protein-Protein Interaction (PPI) Networks:
  STRING database: >67 million interactions. Y2H screens; AP-MS; literature mining.
  Network topology: Hubs (highly connected nodes; often essential genes); scale-free networks.
  Topological Data Analysis: Persistent homology to identify network features.

Flux Balance Analysis (FBA):
  Genome-scale metabolic model (GEM) + stoichiometry constraints + objective function
  (e.g., maximise growth) → linear programming → predicts metabolic flux distribution.
  Used to: Predict gene essentiality; design metabolic engineering strategies;
  identify drug targets (RECON2/RECON3D for human; iML1515 for E. coli).

Multi-omics Integration:
  Combine: Genome + epigenome + transcriptome + proteome + metabolome.
  Tools: MOFA (Multi-Omics Factor Analysis); DIABLO (mixOmics); SnapATAC2.
  Applications: Identifying cancer subtypes; drug mechanism of action; biomarker discovery.
  ENCODE Project: Comprehensive map of functional elements across human + mouse genomes.
  GTEx: Tissue-specific eQTLs (expression quantitative trait loci) across 54 tissues.

[FUTURE PERSPECTIVE] Digital biology is merging with AI:
  AlphaFold2 solved protein structure prediction — a 50-year grand challenge.
  AlphaFold3 (2024): Extends to DNA, RNA, ligands, post-translational modifications.
  RoseTTAFold All-Atom: Similar capability for biomolecular complexes.
  Generative AI for biology: Protein design (ProteinMPNN, RFdiffusion, ESM-Design);
  molecular generation (diffusion models for drug discovery); biological circuit design.
  Foundation models for biology: Models trained on DNA/RNA/protein will likely revolutionise
  genomics as GPT-type models revolutionised NLP — enabling prediction of function,
  regulation, and disease from sequence alone.
